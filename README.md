# MonitorIsland

</div>

## 简介
这是一个 [ClassIsland](https://github.com/ClassIsland/ClassIsland) 插件，提供了一个`监控`组件，它可以显示您电脑当前的一些信息。  
目前支持的有：  
- 内存使用量
- CPU 利用率
- CPU 温度
